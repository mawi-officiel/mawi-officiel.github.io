Mawi Cache Killer 1.1.0

  • Overview:

This plugin is the perfect tool for web developers, designers, and users experiencing page refresh issues due to browser cache. Mawi provides a radical solution to force your browser to load fresh content directly from the server and ignore older, locally stored files.

lightbulb

Why do you need this plugin?

When developing or browsing websites, browsers often retain older versions of CSS, JavaScript, or images to speed up browsing, preventing you from seeing new updates. This plugin completely eliminates this problem without the need to manually clear the cache repeatedly.

Key Features:

Instant One-Click Disable:

A simple on/off button to disable caching for all websites or specific sites.

Flexible Control Modes (Smart Modes):

All Sites: To disable caching completely.

Include List: To enable the plugin only for specific sites (whitelist).

Exclude List: To run the extension everywhere except for specific locations.

data_array

Developer Friendly:

  • Full support for localhost and port ignoring.
  • Support for IP addresses (e.g., 192.168.x.x).
  • Support for wildcards (*) for highly flexible subdomain control.
  • ⏱️ Advanced and Automatic Options: Option to configure periodic cache clearing at specified intervals (minutes or hours) to automatically keep your browser clean.
  • ⚙️ Easy Management: Modern user interface (Dark Mode) with the ability to import and export domain lists for easy transfer of settings between devices.

  • Frontend Developers: To see code and design changes instantly.
  • QA Testers: To ensure they are testing the latest version of the application.
  • For the average user: To solve problems with website overlap and the failure to display images or new updates.
