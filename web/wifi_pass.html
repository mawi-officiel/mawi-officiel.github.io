Wordlist Setup


Add `.txt` files containing passwords
The tool automatically loads all `.txt` files from the directory
WPS PIN Database:
`pins.csv`: Standard PIN database for quick attacks
`pins_all.csv`: Comprehensive PIN database for thorough testing

You must have these files for the tool to work.


Go to the wordlist folder and place the txt files in the password list. The program will use all the txt files inside the wordlist folder.


Inside the blacklist folder, place the files you don't want the program to use.
